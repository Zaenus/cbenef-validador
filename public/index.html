✅ Validador de cBenef (Código de Benefício Fiscal)

1. Validar um código cBenef

2. Atualizar Tabela Oficial (SEFAZ)

3. Validar Produtos da Empresa (em massa)

Envie uma planilha (.xlsx ou .csv) contendo as colunas:
codigo_produto, nome_produto, ncm, cfop, cst, cbenef

4. Atribuir cBenef aos Produtos

Envie uma planilha (.xlsx ou .csv) com os produtos da empresa. O arquivo deve conter as colunas:
Código, Descrição, Tribut., Cód. NCM
A coluna Tribut. deve conter o código CST (ex.: 40) ou texto descritivo (ex.: Isento). Colunas opcionais CFOP e CST são reconhecidas quando presentes. Quando há mais de um cBenef possível para o CST, o mapeamento NCM configurado na seção 5 é usado para identificar o código correto; se não houver mapeamento, o sistema recomenda o primeiro código aplicável.

5. Mapeamento NCM → cBenef

Quando há múltiplos cBenef disponíveis para um mesmo CST, o sistema usa este mapeamento para identificar o código correto a partir do NCM (e opcionalmente CFOP/CST) do produto.
Envie uma planilha (.xlsx ou .csv) com as colunas:

  • ncm — código NCM (2 a 8 dígitos; prefixos são aceitos)
  • cbenef — código cBenef correspondente
  • cst (opcional) — restringe o mapeamento a um CST específico
  • cfop (opcional) — restringe o mapeamento a um CFOP específico
  • description (opcional) — observação livre
